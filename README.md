# Tactical Turn-Based RPG Game

A lightweight tactical turn-based RPG developed in **C++** using **SFML 3**.

This project was created as part of a personal engineering portfolio to practice:
- Object-Oriented Programming (OOP)
- Game state management
- Turn-based combat systems
- Automated enemy behaviors
- 2D rendering with SFML

---

## 🎮 Features

- Turn-based combat system
- Multiple character classes:
  - Warrior
  - Mage
  - Monk
- Action points and speed-based turns
- Scripted automated opponents
- Real-time display of character statistics
- Sprite-based 2D rendering
